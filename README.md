# nginx_php
Tutorial inicial Docker, Nginx, PHP, MySQL

# Pré-requisito
* Docker
* Docker-compose
* Git

# Executando no Windows

mkdir C:\Desenv
cd C:\Desenv
git clone https://github.com/felipess19/nginx_php.git
cd nginx_php
docker-compose -d up --build
